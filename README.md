# Прогнозирование эффективности химических соединений против вируса гриппа

## Описание проекта

В рамках проекта решается задача анализа и прогнозирования эффективности химических соединений против вируса гриппа на основе набора молекулярных дескрипторов.

Для каждого соединения представлены:

- IC50 — концентрация, необходимая для подавления вирусной активности на 50%;
- CC50 — концентрация, вызывающая гибель 50% клеток;
- SI (Selectivity Index) — индекс селективности, характеризующий соотношение эффективности и токсичности соединения.

Также для каждого объекта предоставлен набор химических дескрипторов

Цель проекта — исследовать данные, выявить закономерности и построить модели машинного обучения для прогнозирования биологической активности соединений.
